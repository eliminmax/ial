# `ial-core`

Crate containing a handful of structs and enums that are to be treated as internal to `ial`, but are needed within `ial-ast`.

Crates other than `ial-ast` and `ial` itself should depend on `ial`'s re-exports from this crate.

Part of [`ial`](https://github.com/eliminmax/ial).

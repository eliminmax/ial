# `ial-ast`

Defines an Abstract Syntax Tree for an Intcode Assembly Language, used within the `ial` crate. It's separated out in order to allow `ial` to maintain strong SemVer guarantees while also allowing the AST implementation to evolve as needed.

Part of [`ial`](https://github.com/eliminmax/ial).
